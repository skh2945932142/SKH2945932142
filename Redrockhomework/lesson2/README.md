# Lesson 2 - CSS布局与样式练习

## 学习内容

本次作业演示了以下CSS概念：

### 1. 盒模型（Box Model）
- 使用了 `box-sizing: border-box` 来统一盒模型计算方式
- 展示了内边距（padding）、边框（border）、外边距（margin）的使用

### 2. Flexbox布局
- 导航栏使用了 `display: flex` 进行水平布局
- 卡片容器使用了 `flex-wrap` 实现响应式布局
- 使用了 `gap` 属性控制元素间距

### 3. 定位（Positioning）
- 演示了相对定位（relative）和绝对定位（absolute）的配合使用
- 展示了定位属性如 `top`、`right` 的效果

### 4. 伪类选择器
- `:hover` - 鼠标悬停效果
- `:focus` - 输入框获得焦点效果
- `:nth-child(odd)` - 选择奇数项

### 5. 过渡动画（Transition）
- 按钮和卡片的hover效果使用了 `transition`
- 实现了平滑的颜色、位置、阴影变化

### 6. 渐变（Gradient）
- 页面头部使用了线性渐变背景 `linear-gradient`

### 7. 响应式设计
- 使用了媒体查询 `@media` 适配移动设备
- 在小屏幕上卡片自动变为单列布局

### 8. 表单样式化
- 自定义了输入框、文本域、按钮的样式
- 添加了获得焦点时的视觉反馈

## 功能特性

1. **平滑滚动**: 点击导航链接可以平滑滚动到对应区域
2. **交互反馈**: 按钮点击有弹窗提示
3. **视觉效果**: 
   - 卡片悬停时会上浮并增强阴影
   - 列表项悬停时会向右移动并改变颜色
   - 输入框获得焦点时有蓝色边框提示

## 文件说明

- `index.html` - 主HTML文件，包含页面结构
- `style.css` - CSS样式文件，包含所有样式定义
- `README.md` - 本说明文档

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件
2. 尝试与页面交互：
   - 点击导航栏链接
   - 悬停在卡片和列表项上查看效果
   - 在表单中输入内容
   - 调整浏览器窗口大小查看响应式效果

## 学习要点

- 理解盒模型的计算方式
- 掌握Flexbox的基本使用
- 学会使用定位创建层叠效果
- 掌握常用伪类选择器
- 学会使用过渡创建动画效果
- 了解响应式设计的基本实现
